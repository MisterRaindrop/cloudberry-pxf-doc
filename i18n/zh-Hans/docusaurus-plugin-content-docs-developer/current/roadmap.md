# PXF Roadmap

This page outlines the development direction for Apache Cloudberry PXF. For the latest updates, check the [GitHub repository](https://github.com/apache/cloudberry-pxf) and [community discussions](https://github.com/apache/cloudberry/discussions).

## Active Areas

- **Cloudberry Integration** — continued alignment with Apache Cloudberry features
- **Performance** — optimizations for large-scale data access
- **New Connectors** — expanding the ecosystem of supported data sources
- **FDW Support** — enhancing the Foreign Data Wrapper interface alongside External Tables
- **CI/CD** — improving test coverage and pipeline reliability

## How to Get Involved

- Browse [open issues](https://github.com/apache/cloudberry-pxf/issues) for tasks to pick up
- Join the [Slack community](https://inviter.co/apache-cloudberry) for discussion
- Propose new features via [GitHub Discussions](https://github.com/apache/cloudberry/discussions)
- See [Contributing](./contributing.md) for the contribution workflow
